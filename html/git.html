Перед вами небольшая статья-шпаргалка, которая включает в себя команды удобные, но не всем знакомые команды Git, которые могут стать вашими постоянными спутниками в разработке.

git init

Эта команда используется для инициализации проекта как репозитория git.


git clone https://github.com/Chingiz1337/aboba.git aboba

Эта команда клонирует репозиторий в новую директорию с именем "aboba".

git config — это удобная функция, которая используется для настройки значений конфигурации Git на глобальном и локальном уровнях проекта. Примеры команд:

git commit -m "Commit message"

Эта команда создает новый коммит с сохраненными изменениями и комментарием о проделанной работе.

git status

Эта команда отображает состояние рабочего дерева и индекса, показывая, какие файлы были изменены, добавлены или удалены.

git branch

Эта команда показывает список всех веток в репозитории и указывает текущую активную ветку.
